🗺笔记生成思维导图

此项功能在 2.1.6 及以后提供。

Memocast 使用了优秀的开源项目 Markmap 来实现思维导图功能！我们使用了如下的组件来实现思维导图的生成与预览。

  • markmap-lib
  • markmap-view

我们在笔记中提供了便捷生成脑图的方式，只要你在加载完成的笔记中点击右键，选择生成思维导图即可。

即将提供#

  • 提供工具栏，方便快速缩放到屏幕大小
  • 提供 png，html，svg 导出。
